## 📖 Sobre o Projeto | About the Project

**[PT]** O **PIGMOBI Plan Simulator** é uma aplicação web interativa projetada para simplificar a apresentação e venda de planos de software (SaaS) para restaurantes. O simulador permite que vendedores e clientes escolham planos base, adicionem módulos extras dinamicamente e visualizem um orçamento detalhado em tempo real. O foco é em conversão, clareza de informações e uma experiência de usuário (UX) premium com animações fluidas e design responsivo.

**[EN]** The **PIGMOBI Plan Simulator** is an interactive web application designed to streamline the presentation and sales of software plans (SaaS) for restaurants. The simulator allows sales reps and clients to choose base plans, dynamically add extra modules, and view a detailed quote in real-time. The focus is on conversion, information clarity, and a premium user experience (UX) featuring fluid animations and a responsive design.

## ✨ Funcionalidades | Features

- **Simulador Interativo em Tempo Real:** Cálculo imediato de mensalidades e custos de setup conforme a seleção de planos e adicionais.
- **Tabela Comparativa de Planos:** Visão detalhada e responsiva das diferenças entre os planos (Smart, Prime, Infinity).
- **Design Responsivo & Mobile First:** Interface adaptável, com "Sticky Bottom Bar" no mobile para manter os totais e CTAs sempre acessíveis.
- **Micro-interações e Animações:** Transições suaves utilizando Framer Motion, elevando o valor percebido do produto.
- **Exportação e Compartilhamento:** Geração de propostas amigáveis para impressão (PDF) e integração direta com WhatsApp.
- **Layout "Slop Free":** Estrutura compacta, minimizando a necessidade de rolagem vertical.

## 🚀 Como Executar | Setup Instructions

### Pré-requisitos
- Node.js (v18+)
- npm ou yarn

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/HenriqueHintz/Planso-PIGMOBI.git
   ```

2. **Acesse o diretório:**
   ```bash
   cd Planso-PIGMOBI
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse a aplicação:**
   Abra `http://localhost:3000` no seu navegador.
